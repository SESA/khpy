khpy

**K**ittyhawk in Python

Set Kittyhawk platform type and db path in /kh.cfg


###Commands: 

alloc
clean 
console 
info
init
install

remove


###Requirements: 
python 2.x 
python daemon library:
  https://pypi.python.org/pypi/python-daemon/
