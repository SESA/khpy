khpy
=======

Kittyhawk in Python

A client-server implementation of Kittyhawk

###Instructions: 
Set Kittyhawk platform type and db path in /kh.cfg


###Commands: 

alloc  
clean   
console   
info  
init  
install  
network  
remove  


###Requirements: 
python 2.7  

#####Qemu modules requirements 
qemu-kvm  
brctl  
dnsmasq  
tunctl  

### Install (ubuntu):
cp khpy/initd /etc/initd/khpy
